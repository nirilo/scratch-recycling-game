# ♻️ Bin It Right! — Scratch Recycling Game

This project is a fun educational game created with Scratch.  
The player must catch falling waste and sort it into the correct bin before it reaches the ground.

## 🎮 How to Play

- **Left / Right Arrow** — Move the bin
- **1** — Green bin for general waste
- **2** — Brown bin for organic waste
- **3** — Blue bin for recycling

Correct sorting gives **+1 point**, while an incorrect choice gives **−1 point**.

## 🗑️ Waste Categories

- 🔵 **Blue Bin:** paper, plastic, glass, aluminium, and recyclable packaging
- 🟤 **Brown Bin:** food waste, fruit peels, vegetables, and other organic waste
- 🟢 **Green Bin:** general non-recyclable waste

## 🚀 Game Features

- Random falling waste objects
- Score and sound effects
- Increasing speed!!!
- Keyboard-controlled bin selection
- Animated **WIN** sequence
- Educational recycling practice : check!

## 🛠️ Built With

- Scratch
- Custom and edited PNG images
- Sprites, costumes, clones, variables, sounds, and broadcast messages

## ▶️ Run the Project

1. Download the `.sb3` file from the `project` folder.
2. Open Scratch and select **File → Load from your computer**.
3. Choose the project file.
4. Press the green flag and start sorting!
